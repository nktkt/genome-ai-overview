# ゲノム分析AI 用語集

最終更新: 2026-04-18

---

## 生物学基本

| 用語 | 解説 |
| --- | --- |
| **SNV** (Single Nucleotide Variant) | ゲノム上の1塩基のみが置換されたバリアント。SNPは集団中で頻度が高いSNVを指す。 |
| **indel** | 数塩基〜数十塩基の挿入 (insertion) または欠失 (deletion) の総称。フレームシフトを引き起こすことがある。 |
| **CNV** (Copy Number Variation) | 遺伝子やゲノム領域のコピー数が個体間で異なる構造変異。疾患感受性や表現型多様性に関与。 |
| **SV** (Structural Variant) | 50bp以上の大きなゲノム構造変化。逆位、転座、重複、欠失、複雑再編成などを含む。 |
| **MSA** (Multiple Sequence Alignment) | 3本以上の配列を整列させ保存領域や変異部位を比較する手法。進化解析やタンパク質機能予測の基礎。 |
| **UTR** (Untranslated Region) | mRNAのうち翻訳されない領域 (5' UTR/3' UTR)。翻訳制御やmRNA安定性に関与。 |
| **エンハンサー** | 距離やオリエンテーションによらず転写を促進するシス制御配列。組織特異的遺伝子発現の鍵。 |
| **プロモーター** | 遺伝子の転写開始点近傍に位置し、RNAポリメラーゼや転写因子を動員するシス制御配列。 |
| **エピゲノム** | DNA配列自体を変えずに遺伝子発現を制御する修飾の総体 (メチル化、ヒストン修飾、クロマチン構造など)。 |
| **メチル化** | 主にCpGサイトのシトシンに付加されるDNA修飾。一般にプロモーター領域で転写抑制に働く。 |
| **ヒストン修飾** | ヒストン尾部のアセチル化・メチル化などの化学修飾。クロマチン状態と遺伝子発現を制御。 |
| **クロマチンアクセシビリティ** | DNAが転写因子等にアクセスされやすい度合い。ATAC-seqやDNase-seqで測定される。 |
| **トランスクリプトーム** | 細胞や組織で発現している全RNAの総体。RNA-seqなどで定量される。 |
| **プロテオーム** | 発現する全タンパク質の総体。質量分析等で解析され、ゲノム情報と統合される。 |

## シーケンス技術

| 用語 | 解説 |
| --- | --- |
| **NGS** (Next-Generation Sequencing) | 数百万〜数十億のDNA断片を並列に読む超並列シーケンス技術の総称。 |
| **WGS** (Whole Genome Sequencing) | ゲノム全体を読むシーケンス。コーディング領域に加え非コード領域や構造変異も解析可能。 |
| **WES** (Whole Exome Sequencing) | エクソン領域を濃縮して読むシーケンス。コスト効率よく疾患関連コーディング変異を検出。 |
| **scRNA-seq** | 1細胞単位でトランスクリプトームを測定する技術。細胞異質性や希少細胞集団の解析に必須。 |
| **snRNA-seq** | 単一核からRNAを抽出して解析する手法。凍結組織や神経細胞など単離困難な試料に適する。 |
| **ONT** (Oxford Nanopore) | ナノ孔を電流変化で検出するロングリードシーケンス。リアルタイム読み取りとメチル化同時検出が可能。 |
| **PacBio HiFi** | PacBioの高精度ロングリード (Q30以上, 10〜25kbp)。構造変異解析やゲノムアセンブリで標準化。 |
| **ロングリード** | 1リードが数kbp〜数Mbpに及ぶ長尺配列。リピート領域や構造変異の解像度が高い。 |
| **ショートリード** | 50〜300bp程度の短いリード。Illuminaが代表でコスト・精度・スループットに優れる。 |
| **空間トランスクリプトミクス** | 組織切片上で位置情報を保ったままmRNAを定量する技術 (Visium, Stereo-seq等)。 |
| **ATAC-seq** | トランスポザーゼでオープンクロマチン領域を断片化して読む手法。制御領域のマッピングに使用。 |
| **ChIP-seq** | 特定タンパク質結合DNAを免疫沈降して読む手法。転写因子やヒストン修飾の局在解析に用いる。 |
| **Hi-C** | ゲノムの3次元近接関係を検出する手法。TADやA/Bコンパートメントの同定に使用。 |

## AI / ML

| 用語 | 解説 |
| --- | --- |
| **CNN** (Convolutional Neural Network) | 局所パターンを畳み込みで抽出するニューラルネット。モチーフ検出や配列分類に多用される。 |
| **RNN / LSTM** | 系列情報を状態として伝播する再帰型ネット。配列データの初期モデリングで使われた。 |
| **Transformer** | 自己注意機構で長距離依存を並列処理するアーキテクチャ。ゲノム基盤モデルの主流。 |
| **基盤モデル** (Foundation Model) | 大規模データで事前学習され多様な下流タスクに転用可能な汎用モデル。DNA/RNA/タンパク質用が登場。 |
| **自己教師あり学習** | ラベルなしデータから擬似タスク (マスク予測など) を作って学習する枠組み。生物配列学習の中核。 |
| **トークン化** | 配列を離散単位 (塩基、k-mer, BPEなど) に分割しモデル入力化する処理。 |
| **アテンション** | 入力各要素の関連度を重み付けして情報を集約する機構。Transformerの中核コンポーネント。 |
| **fine-tuning** | 事前学習済みモデルを特定タスク用データで追加学習して適合させる手法。 |
| **ゼロショット** | 当該タスクのラベル付き例なしで推論を行う能力。基盤モデルの転用性評価に用いられる。 |
| **few-shot** | ごく少数の例示のみで推論や適応を行う能力。実験コストの高いバイオ領域で重要。 |
| **LoRA / PEFT** | 少数の追加パラメータのみで大規模モデルを効率適応する手法群。 |
| **埋め込み** (Embedding) | 配列や変異を連続ベクトルへ写像した表現。下流の分類・検索・可視化に用いる。 |
| **拡散モデル** | ノイズ付加と逆過程を学習して生成する確率モデル。タンパク質・DNA配列生成にも応用される。 |
| **グラフニューラルネット** (GNN) | ノードとエッジで構成されたデータを学習する手法。遺伝子制御ネットや相互作用解析に有効。 |
| **説明可能AI** (XAI) | 予測根拠を人が解釈可能な形で提示する技術。臨床応用で必須とされる。 |
| **不確実性推定** | 予測の信頼度を定量化する手法。VUS判定や臨床判断支援で重要。 |

## ゲノムAI固有

| 用語 | 解説 |
| --- | --- |
| **variant calling** | アラインメント結果から個体のバリアント (SNV/indel/SV等) を同定する工程。DeepVariant等AI手法が主流化。 |
| **GRN** (Gene Regulatory Network) | 転写因子-標的遺伝子などの制御関係ネットワーク。scRNA-seqとAIで大規模推定が進む。 |
| **PRS** (Polygenic Risk Score) | 多数の遺伝バリアントの効果を統合した疾患リスクスコア。AIで非線形効果も取り込む研究が進行。 |
| **in silico 摂動** | モデル内で遺伝子や変異を仮想的に操作し応答を予測する解析。GeneformerやscGPTで普及。 |
| **de novo 設計** | 既存配列に依存せず目的機能を満たす配列を新規生成する設計手法。タンパク質や制御配列で実用化。 |
| **SaMD** (Software as a Medical Device) | それ自体が医療機器として規制対象となるソフトウェア。AI診断ツールが該当する。 |
| **VUS** (Variant of Uncertain Significance) | 病原性の評価が定まらないバリアント。AIによる再分類支援が活発。 |
| **ACMG** ガイドライン | 米国臨床遺伝学会が定める変異病原性分類基準 (PVS1, PS, PM, PP, BP等)。臨床解釈の世界標準。 |
| **splice prediction** | スプライス部位や異常スプライシングを予測するタスク。SpliceAI等の深層学習が代表的。 |
| **効果予測** (variant effect prediction) | 変異の機能影響を予測するタスク。AlphaMissenseやESM系モデルが広く利用される。 |
| **eQTL** | 遺伝子発現量と関連するゲノム多型。AIで組織横断的に統合推定する研究が盛ん。 |
| **細胞アトラス** | 全組織・全細胞タイプの包括的単一細胞参照地図。基盤モデル学習の主要コーパス。 |
| **バーチャルセル** | 細胞の状態・応答をシミュレートするAIモデル。摂動応答予測や創薬に応用。 |
| **タンパク質構造予測** | 配列から立体構造を予測するタスク。AlphaFoldを起点に設計・機能予測に展開。 |
| **多モーダル統合** | ゲノム、トランスクリプトーム、エピゲノム、画像などを統合学習する枠組み。 |
| **ベンチマーク** (例: GUE, BEND) | ゲノム基盤モデルの性能比較用標準タスク群。公平評価のための共通指標。 |
| **データリーク** | 学習データと評価データが相同配列等で重複する問題。ゲノムでは配列類似性に基づく分割が必須。 |
| **プライバシー保護学習** | フェデレーテッド学習や差分プライバシー等、個人ゲノム保護を伴う学習。臨床AIで重要。 |

## 略語早見

- **SNV/SNP**: 単塩基変異 / 多型
- **CNV/SV**: コピー数変異 / 構造変異
- **WGS/WES**: 全ゲノム / 全エクソーム
- **NGS/ONT**: 次世代シーケンス / ナノポア
- **GRN/PRS**: 遺伝子制御ネット / 多遺伝子リスクスコア
- **VUS/ACMG**: 意義不明変異 / 米国臨床遺伝学会
- **SaMD**: 医療機器ソフト
